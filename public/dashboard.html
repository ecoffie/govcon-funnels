How Lead Capture Works

All forms use the LeadForm component which posts to /api/lead:

  1. User fills out form (name, email, phone)
  2. Form posts to /api/lead with source tag
  3. API creates/updates contact in Go High Level CRM
  4. API sends Slack notification to team channel
  5. API sends confirmation email (if configured)
  6. User redirects to thank you/upsell page

üß™ Testing Lead Automation

To test if leads are flowing correctly:

  1. Fill out any form with test data (use unique email like test-{source}@example.com)
  2. Check Slack channel for notification (should appear within seconds)
  3. Check Go High Level for new contact with correct source tag
  4. Check email inbox for confirmation (if email configured for that source)
